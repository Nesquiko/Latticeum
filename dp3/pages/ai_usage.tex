\chapter*{Use of AI}
\addcontentsline{toc}{chapter}{Use of AI}
\markboth{Use of AI}{Use of AI}

AI was used in document writing, implementation, and evaluation.

\section{Document writing}

\begin{itemize}
	\item brainstorming how to best write and explain certain parts,
	\item proofreading,
	\item cross-checking citations,
	\item writing LaTeX commands and TikZ diagrams.
\end{itemize}

This was controlled by reading the generated corrections, opening the cited papers
at the reported places, and checking the compiled document and diagrams visually.

\section{Implementation}

\begin{itemize}
	\item debugging CCS satisfiability in SageMath,
	\item writing tests for correct RISC-V VM execution and instruction decoding,
	\item navigating the forked dependencies,
	\item writing VM instruction handlers,
	\item setting up the EVM guest program for proving block 24740084,
	      including downloading all necessary block, transaction, and Ethereum state data
	      from the Ethereum blockchain.
\end{itemize}

This was controlled by changing constraints until the numbers in the constraints
matched the numbers from the Sage scripts and the checks passed, running the
tests on compiled guest binaries to validate the VM instruction handlers,
opening the reported dependency sources to verify that the guidance matched the
code, and running REVM on the downloaded data to validate block 24740084.

\section{Evaluation}

\begin{itemize}
	\item researching how other zkVMs are run.
\end{itemize}

This was controlled by actually running the referenced systems and checking that
the described commands and observed behaviour matched.
